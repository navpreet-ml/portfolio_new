Air Quality Sensors

Working on designing a MEMS based particulate matter sensor. The end goal is to fit all the sensing technology and the electronic circuitary in a 20x20x15 mm3 package. The sensor uses the frequency response of the resonator to measure the amount of matter in the air.

Weather Alerts Script

Scripted a daily weather alert using python. The script uses a weather and a SMS API to read weather for the next 8 hours and send a text to my cellphone if it is going to rain (reminder to take umbrella). The script is hosted on PythonAnywhere for autorun at 6 am everyday.

Text-to-Speech Desktop App

Implemented an installable desktop app to convert PDF to MP3 using Google text-to-speech. The app was implemented using Tkinter library with a button to upload the PDF, a convert button to start converting and a download button to download the converted MP3.

Automated Data Entry
